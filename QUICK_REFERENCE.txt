# QUICK REFERENCE CARD - Disease Prediction System

## 🚀 Quick Start (3 Steps)

### Step 1: Validate Dataset
```bash
python setup_dataset.py
```
Expected: ✓ Dataset validated successfully

### Step 2: Train Models (20-30 min CPU, 5-10 min GPU)
```bash
python train_improved.py
```
Expected: ✓ Models saved to models/ directory

### Step 3: Run Application
```bash
python app.py
```
Expected: ✓ Flask running on http://localhost:5000

## 🐛 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| ModuleNotFoundError | Run: `pip install -r requirements_improved.txt` |
| Model not found | Run: `python train_improved.py` |
| Low accuracy | Check image quality, ensure clear leaves/berries |
| All uncertain | Image too blurry, upload clearer image |
| Crop misclassified | Dataset issue, retrain or update model |
| Memory error | Reduce batch size in config |
| Slow inference | Use GPU or reduce batch size |

## 🎓 Key Concepts

### Two-Stage Prediction:
1. **Stage 1:** Run image through both crop models
2. **Stage 2:** Compare confidence scores
3. **Stage 3:** Select model with highest confidence
4. **Stage 4:** Get disease information

### Confidence Threshold:
- < 70%: "Uncertain - upload clearer image"
- ≥ 70%: Valid prediction with full details

### Transfer Learning:
- Uses pre-trained ImageNet weights
- Saves training time
- Improves accuracy with limited data
